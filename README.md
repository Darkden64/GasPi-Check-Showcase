<h2>🏗 Architecture Standard</h2>
<ul>
  <li><b>Frontend</b> : React Native / Expo / TypeScript (Strong Typing).</li>
  <li><b>Cloud Sync</b> : Firebase Firestore Real-time (Family Sync jusqu'à 6 membres).</li>
  <li><b>Local-First</b> : Architecture hybride AsyncStorage/Cloud pour une performance offline-first.</li>
</ul>

<h2>🎮 Gamification & Behavioral Science</h2>
<ul>
  <li><b>Économie Hardcore</b> : Système punitif/récompensant (0 XP ajout, -200 XP gâchis, -100 XP périmé) pour un changement de comportement réel.</li>
  <li><b>Grades Zen-Eco</b> : 10 paliers de progression avec cartes de pouvoirs stratégiques (Turbo XP, Reset).</li>
  <li><b>Statistiques</b> : Bandeau blanc arrondi avec trois colonnes (Gain XP, Perte XP, Score d'efficacité) et vue hebdomadaire par défaut.</li>
</ul>

<h2>🛡 Privacy & RGPD Compliance</h2>
<ul>
  <li><b>Kill Switch (V9.5)</b> : Suppression instantanée et totale des données (Cloud & Local).</li>
  <li><b>Anonymisation</b> : IDs rotatifs et Data Minimization (Zéro tracking nominatif).</li>
</ul>

<h2>📊 Impact RSE</h2>
<ul>
  <li><b>Barcode Analytics V2</b> : Insights sur le gaspillage post-achat par EAN.</li>
  <li><b>Équivalences CO2</b> : Traduction de l'impact en mesures concrètes (Vols NY, Arbres).</li>
  <li><b>Sentinelle</b> : Suppression automatique de la corbeille le dimanche à 23h59.</li>
</ul>
